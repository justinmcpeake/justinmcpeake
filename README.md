## What I'm building

**[Fortress](https://github.com/justinmcpeake/fortress-architecture)** — a private AI assistant platform for law firms, medical consultants, and high-net-worth individuals who need AI but can't send client data to the cloud.

- Local inference on UK-controlled hardware — no data leaves the premises
- Multi-agent workflows with human-in-the-loop controls throughout
- Built to UK GDPR / data sovereignty requirements

## Stack

```
Orchestration   →  n8n + CrewAI

Vector store    →  ChromaDB + LlamaIndex
Embeddings      →  Nomic-Embed-Text
RAG             →  LlamaIndex document pipelines
Voice           →  Piper TTS + Whisper.cpp
```
